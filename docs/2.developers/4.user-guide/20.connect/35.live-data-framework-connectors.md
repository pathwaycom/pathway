# Pathway Live Data Framework connectors

The Pathway Live Data Framework has a long list of connectors for connecting to external data sources at input and connectors for outputting the data outside of the framework.
Since Pathway Live Data Framework is a streaming-first framework, **all connectors operate in streaming mode**, automatically updating the results in real-time with every change.
